A simple and responsive registration form built using React.
It includes real-time validation, error handling, and a clean UI for collecting user details such as name, email, phone, PAN, Aadhaar, and more.

Features:
1. Validation for all fields
2. Show/Hide password option
3. Navigation to details page
4. Fully reusable form components
